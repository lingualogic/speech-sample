Eine Website oder Web-App erweitert um Sprachdienste

Dieses Beispiel nutzt Speech-Angular,
um eine Website oder Web-App um ein Voice User Infaces (VUI) zu erweitern.


Features:

Cloud

In Cloud werden die Credentials des jeweiligen Cloud-Dienstes eingetragen, um den Cloud-Dienst verwenden zu können.

Speak

In Speak wird Sprachausgabe mittels Sprachsynthese und durch Verwendung von Audiodateien vorgestellt.

Action

In Action wird gezeigt, wie man die Funktionalität der App nutzen und steuern kann.

Bot

Im Bot können Dialoge designt und ausführt werden, so lassen sich eigene VUI Szenarien entwickeln.

Listen

In Listen wird Spracherkennung mittels der Web Speech API vorgestellt. Spracheingaben können lokal gespeichert werden.

Intent

In Intent wird die Sprachanalyse vorgestellt.
