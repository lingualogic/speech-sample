Google Dialogflow

Der Cloud-Dienst Google Dialogflow von Google Cloud ist ein Assistent-Service, der mit Hilfe Deep Learning-Technologien Sprache versteht und zu Sätzen die passenden Intents zurückgibt. Er erweitert Intentservice um eine zusätzliche NLU, die zur Sprachanalyse genutzt werden kann. Die Nutzung des Cloud-Serices setzt einen Dialogflow Account voraus.

zur Dialogflow-Seite

Die Google Cloud bietet darüber hinaus Dienste für Spracherkennung und Sprachsynthese an. Die Cloud Speech-to-Text API erweitert den Listenservice um eine ASR. Die Cloud Text-to-Speech API erweitert den Speakservice um eine TTS mit weiteren Stimmen. Die Aktivierung der APIs sowie die Verwaltung der Zugriffsrechte kann in der Google Cloud Konsole vorgenommen werden.

Credentials setzen:

Zur Nutzung der Cloud Dienste wird ein client access token benötigt. Der Token wird serverseitig oder via FaaS erzeugt. Ein Beispiel für einen Server bildet der Speech-Tokenserver. Die URL für des Servers wird abhängig von der Nutzung für Dialogflow oder ASR und TTS gesetzt. Für Dialogflow wird darüberhinaus die Project ID angegeben. Eine Verbindung zu Dialogflow kann auch über den App Key hergestellt werden. (deprecated) Der App Key kann in den Setting des Dialogflow Assistenten im Tab General unter API Keys eingesehen werden.
