Rasa Dialogflow

Der Rasa Cloud-Dienst ist ein Assistent-Service, der mit Hilfe Deep Learning-Technologien Sprache versteht und zu Sätzen die passenden Intents zurückgibt. Er erweitert Intentservice um eine zusätzliche NLU, die zur Sprachanalyse genutzt werden kann. Die Nutzung des Cloud-Serices setzt einen aktiven Rasa-Server voraus.

Zur Nutzung von Rasa wird ein Client access token benötigt. Das Token wird beim Start des Rasa-Servers übergeben. Das gleiche Token muss unten als Client access token eingetragen werden. Er wird lokal im Browser gesetzt und ausschließlich an den Rasa-Server übermittelt.
